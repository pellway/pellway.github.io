Keras Mushroom Model

Deep Learning

This project was developed in my spare time during work, where I endeavored to learn more about the practical application of machine learning, as I had studied the subject numerous times at university but with alot of research focus.

I first started with finding an interesting dataset and settled on a "poisonous-or-not" mushroom dataset, where each entry would contain various physical properties of a particular mushroom, such as the number of gills or the colour of the cap.

This project is a binary classification machine learning model that determines whether a mushroom is edible or not. I used Python for all the code, and the Keras library to assist with implementing the machine learning model.

Link to the project's source code can be found here, and the dataset used here.
